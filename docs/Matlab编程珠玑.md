###### 其它函数

1.deal函数

```
% 假如要初始化A,B,C和D为3*4阶矩阵，平常我们会用如下代码
A = zeros(3,4);
B = zeros(3,4);
C = zeros(3,4);
D = zeros(3,4);

% 利用deal()改造后,简洁相当多
[A,B,C,D] = deal(zeros(3,4));
```
